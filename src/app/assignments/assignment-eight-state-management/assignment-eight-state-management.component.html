Assignment 7. Boats, boats boats!

Create a state management system for the requirements below

A client has a business running rental boats. He wants to be notified whenever a boat is rented

Make sure to include the following functionality

  1. A boat has an Id, a name, a flag whether it's available to rent and a type (sailboat, speedboat etc.)
  2. When the client navigates to the landing page, he wants to see all of his available boats
  3. The client should have the option to see either all boats, or the available boats only.
  4. The clients wants to be able to add new boats and remove existing ones.
  5. After adding or removing, the latest changes need to be pushed to the screen
  6. A boat should have a button: Rent. When a boat is rented, it should no longer be visible on the page,but must still exist in the state management. See above
  7. When a boat is rented, A message should be logged to the console, containing the boat details
  8. When a client is trying to rent an unavailable boat, an error message should be displayed
  9. This entire assigment can be done in application state. Data does not need to be persisted
  10. Optional: Can you add styling using scss?
